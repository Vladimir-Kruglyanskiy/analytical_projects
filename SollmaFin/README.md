# Анализ A/B-теста: влияние нового онбординга в SollmaFin

## Суть проекта

Оценка эффективности обновленного онбординга в трейдинговом приложении SollmaFin: анализ исторических данных, проверка бизнес-гипотез через A/B-тест, исследование влияния на поведение пользователей.

## Цель проекта

1. Проанализировать поведение новых пользователей на исторических данных
2. Оценить результаты A/B-теста нового онбординга
3. Проверить гипотезы команды продукта
4. Дать рекомендации по внедрению

## Задачи

- Загрузить и предобработать данные
- Проанализировать динамику привлечения и построить воронки событий
- Оценить влияние уровня риска активов на повторные депозиты
- Рассчитать ARPU на исторических данных
- Сравнить воронки контрольной и тестовой групп
- Проверить гипотезы с помощью t-test и z-test
- Проанализировать ключевые метрики (ARPU, конверсия, ARPPU)
- Провести бутстрап-анализ перцентилей депозитов

## Данные

В работе использованы 2 датасета:

| Таблица | Период | Описание |
|---------|--------|----------|
| `pa_sollmafin_hist.csv` | 01.04 – 01.06.2025 | Исторические данные о поведении пользователей |
| `pa_sollmafin_abt.csv` | 02.06 – 15.06.2025 | Данные A/B-теста |

**Основные поля:** `user_id`, `country_code`, `platform`, `event_name`, `amount`, `asset`, `risk_level`, `group` (только в ABT)

## Инструменты и технологии

- **Python** (pandas, numpy, matplotlib, scipy, statsmodels)
- **Jupyter Notebook**
- **Статистические тесты** (t-test, z-test, бутстрап)

## Ключевые выводы

### 1. Исторические данные
- Ежедневный приток: 620–710 пользователей
- Основные потери — после онбординга и между первым и вторым депозитом
- Конверсия в повторный депозит: 26.36% (high risk) vs 46.41% (low risk)
- ARPU = 91.56 у.е.

### 2. Результаты A/B-теста
- Группы сбалансированы (контроль: 4 847, тест: 4 568)
- В тестовой группе конверсия в онбординг ниже, но последующие этапы — лучше
- Высокорисковые активы: прирост конверсии во 2-й депозит +18.34 п.п. (стат. значимо, p < 0.001)
- ARPU +2.22 у.е. (не значимо, p = 0.331)
- Конверсия в 1-й депозит -1.51 п.п. (не значимо, p = 0.121)
- Конверсия 1→2 депозит +12.35 п.п. (значимо, p < 0.001)
- ARPPU +5.90 у.е. (значимо, p = 0.035)

### 3. Бутстрап-анализ (платящие пользователи)
- 25-й перцентиль: тестовая группа вносит меньше (ДИ: [-50, -44])
- 50-й перцентиль: аналогично (ДИ: [-28, -19])
- 75-й перцентиль: тестовая группа вносит больше (ДИ: [11, 29.5])

### 4. Проверка гипотез

| Гипотеза | Результат |
|----------|-----------|
| Рост (обучение → чаще открывают 2-й депозит) | ✅ Подтверждена |
| Риска (информация о рисках → снижение 1-го депозита) | ❌ Не подтверждена |
| Дополнительная (high risk → чаще возвращаются) | ✅ Подтверждена |

## Рекомендации

1. **Внедрить новый онбординг на всех пользователей** — он не навредил конверсии в первый депозит и улучшил ключевые метрики
2. **Доработать сценарий первого депозита** — потенциальная точка роста
3. **Усилить работу с крупными инвесторами** — они стали вкладывать больше
