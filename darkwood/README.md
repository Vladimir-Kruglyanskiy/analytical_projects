# Проект «Секреты Тёмнолесья»

Анализ внутриигровых покупок в онлайн-игре: изучение влияния характеристик игроков и их персонажей на покупку внутриигровой валюты «райские лепестки», оценка активности платящих и неплатящих игроков.

## Задачи

- Определить долю платящих игроков в целом и в разрезе расы персонажа
- Проанализировать стоимость и количество внутриигровых покупок
- Выявить аномальные (нулевые) покупки
- Сравнить активность платящих и неплатящих игроков
- Определить самые популярные эпические предметы
- Выявить зависимость активности игроков от расы персонажа

## Данные

База данных `fantasy` содержит 7 таблиц:

| Таблица | Описание |
|---------|----------|
| `users` | Информация об игроках (id, расы, класс, регистрация, платящий/неплатящий) |
| `events` | Покупки внутриигровой валюты (сумма, дата, предмет) |
| `race` | Справочник рас персонажей |
| `items` | Справочник эпических предметов |
| `classes` | Справочник классов персонажей |
| `skills` | Справочник легендарных умений |
| `country` | Справочник стран регистрации |

## Инструменты и технологии

- **SQL** (PostgreSQL)
- **DBeaver** - среда для работы с базой данных

## Рекомендации

1. **Балансировка рас** - снизить разрыв между расами, взяв за эталон Northman и Human
2. **Внимание на Demon** - при низких показателях активности, доля платящих самая высокая. Балансировка этой расы может увеличить прибыль
3. **Акции с нулевыми покупками** - расширить практику (если это действительно акции), чтобы стимулировать дальнейшие покупки
