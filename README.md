## Структура репозитория

### [`darkwood`](https://github.com/Vladimir-Kruglyanskiy/analytical_projects/tree/main/darkwood)

**Проект:** Онлайн-игра (Анализ внутриигровых покупок)

**Инструменты:** SQL (PostgreSQL)

**Краткое описание:** Анализ влияния характеристик игроков и их персонажей на покупку внутриигровой валюты «райские лепестки». Оценка активности платящих и неплатящих игроков.

**Содержимое папки:**

| Файл | Описание |
|------|----------|
| `Secrets_Darkwood.sql` | SQL-запросы для анализа: доля платящих игроков, статистика покупок, активность в разрезе рас, популярные предметы |
| `README.md` | Детальное описание проекта, ключевые выводы и рекомендации |

---

### [`real_estate`](https://github.com/Vladimir-Kruglyanskiy/analytical_projects/tree/main/estate_market)

**Проект:** Рынок недвижимости Санкт-Петербурга и Ленинградской области

**Краткое описание:** Исследование времени активности объявлений, сезонности публикаций и продаж, сравнительная оценка сегментов Санкт-Петербурга и Ленинградской области для выработки стратегии выхода на рынок.

**Содержимое папки:**

| Файл | Описание |
|------|----------|
| `estate market analysis.sql` | SQL-запросы для анализа: сегментация по времени активности, сезонность, анализ населённых пунктов Ленобласти |
| `README.md` | Детальное описание проекта, ключевые выводы и рекомендации |

---

### [`investments and startups`](https://github.com/Vladimir-Kruglyanskiy/analytical_projects/tree/main/investments%20and%20startups)

**Проект:** Исследование рынка венчурных инвестиций

**Краткое описание:** Анализ мирового рынка венчурных инвестиций: динамика финансирования стартапов, распределение по отраслям и категориям, факторы, влияющие на успешность стартапов.

**Содержимое папки:**

| Файл | Описание |
|------|----------|
| `investments.ipynb` | Jupyter Notebook с полным анализом: загрузка и предобработка данных, исследовательский анализ, визуализации, выводы |
| `README.md` | Детальное описание проекта, ключевые выводы и рекомендации |
