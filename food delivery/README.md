# Дашборд «Ключевые бизнес-метрики в Саранске»

## Суть проекта

Разработка дашборда с ключевыми бизнес-метриками для сервиса доставки еды: DAU, конверсия, средний чек, LTV и удержание пользователей. Дашборд построен для города Саранск за период май-июнь 2021 года.

## Цель проекта

1. Подготовить SQL-запросы для расчета 5 ключевых метрик
2. Создать визуализации в Yandex DataLens
3. Объединить визуализации в единый дашборд
4. Подготовить аналитическую записку с выводами и рекомендациями

## Метрики на дашборде

| Метрика | Описание |
|---------|----------|
| **DAU** | Ежедневное количество активных пользователей |
| **Conversion Rate** | Коэффициент конверсии из просмотра в заказ |
| **Средний чек (AOV)** | Средняя сумма покупки на пользователя |
| **LTV** | Совокупная ценность клиента за период |
| **Retention Rate** | Коэффициент удержания пользователей |

## Инструменты и технологии

- **SQL** (подготовка данных)
- **Yandex DataLens** (визуализация и дашборд)

## Ключевые выводы

### 1. DAU (активные пользователи за день)
- Наблюдается спад во второй половине июня, метрика находится на уровне ниже среднего за рассматриваемый период
- Количество пользователей растет в будние дни и снижается в выходные
- Метрика выглядит нестабильной

**Возможные причины:** сезонность, праздничные дни в мае и июне, активность конкурентов, маркетинговые кампании

**Рекомендации:** провести акции на выходные для повышения вовлеченности; в качестве амбициозной цели - рассмотреть доставку продуктов из магазинов для расширения клиентской базы

### 2. Conversion Rate (коэффициент конверсии)
- График волатильный: после пиковых значений идет резкий спад, после спада - постепенный «разгон»
- В целом показатели конверсии выглядят стабильными, для этой метрики характерна высокая вариативность

**Возможные причины:** акции, рекламные кампании, праздники; пользователи заказывают любимые блюда, но после нескольких раз они приедаются

**Рекомендации:** внедрить рекомендательную систему для подбора разнообразных блюд, чтобы пользователи пробовали новое и расширяли круг любимых блюд

### 3. Средний чек
- Средний чек, количество заказов и выручка выросли
- В июне количество заказов увеличилось на фоне снижения DAU - это может указывать на рост числа повторных заказов

**Возможные причины:** повышение цен, рост спроса на дорогие блюда, успешные акции в июне

### 4. Retention Rate (коэффициент удержания пользователей)
- На второй день (day_since_install = 1) резкое падение - типичная ситуация для подобных сервисов
- Стабилизация на уровне 4-5%
- Майские когорты показывают лучшее удержание, чем июньские

**Рекомендации:** внедрить накопительную систему бонусов за действия в приложении (просмотр блюд, заказы) для повышения лояльности

### 5. Топ-3 ресторана по LTV и популярные блюда

| Ресторан | LTV |
|----------|-----|
| Гурманское наслаждение | 170 479,19 ₽ |
| Гастрономический шторм | 164 508,16 ₽ |
| Шоколадный рай | 61 199,76 ₽ |

- Значительный разрыв между 2-м и 3-м местом
- Самыми популярными являются мясные блюда
- Отмечено несоответствие названий и состава блюд

## Рекомендации

**Приоритетное направление:** стабилизация DAU

Сейчас основная часть DAU - это новые пользователи, а постоянных мало. DAU нестабильна, общее число пользователей невысокое. Поэтому в первую очередь стоит работать над стабилизацией DAU, и уже после этого — над удержанием и конверсией.

**Конкретные шаги:**

1. **Акции на выходные** - запустить целевую акцию для повышения вовлеченности пользователей в выходные дни (например, скидка на первый заказ в выходной день или специальное предложение «закажи на выходные — получи бонус»)

2. **Рекомендательная система** - разработать механизм рекомендаций на основе истории заказов пользователя, чтобы расширять круг любимых блюд и снижать волатильность конверсии

3. **Накопительная система бонусов** - внедрить программу лояльности, где баллы начисляются не только за покупки, но и за действия в приложении (просмотр блюд, заходы)

4. **Расширение мясных блюд** - заключить партнерства с ресторанами, специализирующимися на мясной кухне, так как мясные блюда — самые популярные

**Стратегическая цель:** рассмотреть возможность запуска доставки продуктов из магазинов для привлечения пользователей, которые предпочитают готовить самостоятельно
