# Поиск инсайтов и точек роста для сервиса доставки еды

## Суть проекта

Анализ пользовательского поведения и партнерской сети сервиса доставки еды для поиска точек роста: выявление проблемных этапов в пользовательском пути, оценка лояльности клиентов через RFM-сегментацию и разработка персонализированных стратегий для ресторанов-партнеров.

## Цель проекта

1. Изучить пользовательский путь и проанализировать конверсии через воронки
2. Сегментировать пользователей с помощью RFM-модели
3. Кластеризовать рестораны методом K-Means
4. Разработать персонализированные стратегии для удержания пользователей и улучшения взаимодействия с партнерами

## Задачи

- Загрузить и предобработать данные
- Построить классическую и step-by-step воронки
- Провести сравнительный анализ воронок для мобильной и десктопной версий
- Рассчитать RFM-метрики и выделить сегменты пользователей
- Провести кластеризацию ресторанов
- Сформулировать выводы и рекомендации

## Ключевые выводы

### 1. Анализ пользовательского пути
- **Мобильное приложение:** проблемные зоны - подтверждение номера телефона (потеря почти 50% пользователей) и страница блюда (падение конверсии на 27%)
- **Десктопная версия:** нет критических просадок на отдельных этапах (потери 10–20%)
- На последних шагах (`add_to_cart` и `order`) у десктопа конверсия ниже, чем у мобильной версии

### 2. RFM-сегментация пользователей

| Сегмент | Доля | Характеристика |
|---------|------|----------------|
| 211, 111, 311 | ~30% | Редко покупают, низкий доход |
| Лояльные (333, 233, 313 и др.) | ~18% | Активные, высокие траты |
| "Ушедшие" (112, 113, 121, 122, 123) | ~18% | Давно не совершали покупок |

### 3. Кластеризация ресторанов

| Кластер | Характеристика | Рекомендация |
|---------|----------------|--------------|
| Кластер №1 | Средний по всем метрикам | Стимулировать заказы акциями |
| Кластер №2 | Лидеры по заказам и ассортименту | Индивидуальные условия сотрудничества |
| Кластер №3 | Высокий чек, минимальный ассортимент | Расширить ассортимент недорогими блюдами |
| Кластер №4 | Отстающие по всем показателям | Изменить формат работы |
