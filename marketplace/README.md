# Анализ маркетплейса

## Суть проекта

Проведение системного продуктового анализа маркетплейса: оценка ключевых метрик, юнит-экономики, поиск точек роста и проверка гипотез через A/B-тестирование.

## Цель проекта

1. Провести первичный анализ данных
2. Выполнить верхнеуровневую оценку состояния продукта
3. Оценить метрики монетизации и юнит-экономику
4. Найти инсайты и точки роста через сегментацию
5. Подвести итоги A/B теста

## Задачи

- Загрузить и предобработать данные (Users, Events, Orders, Campaign_costs)
- Проанализировать динамику DAU, WAU, MAU, Revenue, ARPU, ARPPU, AOV, Retention, CR
- Рассчитать CAC, LTV, ROI, Payback Period
- Провести сегментацию по каналам, типам пользователей, регионам, устройствам
- Сформулировать гипотезы для развития продукта
- Проанализировать A/B-тест

## Данные

В работе использованы 4 основных датасета:

| Таблица | Описание |
|---------|----------|
| `Users` | Информация о пользователях (регистрация, возраст, пол, регион, канал привлечения, сегмент) |
| `Events` | События пользователей (просмотры, клики, добавление в корзину, оформление заказа и др.) |
| `Orders` | Информация о заказах (товары, количество, цена, категория) |
| `Campaign_costs` | Затраты на маркетинг по каналам и месяцам |

## Инструменты и технологии

- **Python** (pandas, matplotlib, seaborn, numpy)
- **Jupyter Notebook**
- **Статистические тесты** (t-тест, бутстрап)

## Ключевые выводы

### 1. Общая динамика метрик
- Активность пользователей (DAU, WAU, MAU) растет до августа, с сентября начинается снижение
- Выручка снижается с сентября месяц к месяцу
- Удержание (Retention) падает с каждой новой когортой: если в первых когортах на 2-й месяц возвращалось 30-35%, то в последних - ниже 25%
- Конверсия в заказ начинает снижаться уже с июня

### 2. Юнит-экономика
- ARPU падает с августа
- Затраты на привлечение (CAC) растут с начала года до августа, с сентября снижаются
- LTV растет с начала года, но снижение начинается с мая, и уже в сентябре затраты превышают доход
- Когорты январь-июнь окупаются на 2-й месяц; с августа окупаемость переносится на 3-й месяц

### 3. Сегментация и инсайты
- **Воронка:** узкие места на этапах оформления заказа (после добавления в корзину только 82% начинают оформление, 83% завершают, 64% покупают)
- **Каналы привлечения:** TikTok привлекает больше всего пользователей, но они приносят самый низкий доход; SEO и Affiliate - наиболее перспективные каналы
- **Сегменты пользователей:** 37,2% - средние пользователи, 30% - редкие, 17,6% - регулярные, 15,2% - одноразовые
- **TikTok:** большая часть пользователей отваливается после регистрации

### 4. Результаты A/B-теста (бонус на первую покупку для TikTok)
- Конверсия в первую покупку: контрольная группа — 1%, тестовая — 8% (p-value < 0.05)
- ARPU вырос на 942% (статистически значимо)
- ARPPU вырос (статистически значимо)
- AOV вырос, но изменение не статистически значимо
- Общая выручка выросла на 916%

**Вывод по A/B-тесту:** бонус на первую покупку для пользователей TikTok показал статистически значимый рост ключевых метрик. Рекомендовано распространить бонус на всех пользователей из этого канала.
